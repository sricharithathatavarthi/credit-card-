# 💳 Credit Card Fraud Detection

## 📌 Overview
This project implements a **machine learning system** to detect fraudulent credit card transactions using the Kaggle dataset. Fraud detection is a critical challenge due to the **highly imbalanced nature of the data** (fraud cases are less than 0.2% of all transactions).  

The repository contains code for **data preprocessing, model training, evaluation, and deployment** via a simple Streamlit app.

---

## 📂 Dataset
- **Source:** [Kaggle Credit Card Fraud Detection Dataset](https://www.kaggle.com/mlg-ulb/creditcardfraud)  
- **Size:** 284,807 transactions  
- **Features:**  
  - `Time` and `Amount` (transaction details)  
  - `V1–V28` (anonymized PCA features)  
- **Target:** `Class` (0 = legitimate, 1 = fraud)

---

## ⚙️ Technologies Used
- **Language:** Python  
- **Libraries:**  
  - Data: `pandas`, `numpy`  
  - Visualization: `matplotlib`, `seaborn`  
  - ML Models: `scikit-learn`, `xgboost`  
  - Deployment: `streamlit`
